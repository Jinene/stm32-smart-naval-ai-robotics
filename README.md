# STM32 Smart Naval AI Robotics Platform

## Overview
Autonomous underwater robotics control system using:

- STM32 HAL
- FreeRTOS
- Sensor Fusion
- TinyML anomaly detection
- MQTT Telemetry
- Custom Underwater Protocol

## Architecture
Sensors → Fusion → AI → Communication → Surface Dashboard

## Features
- Deterministic memory pool
- RTOS multi-tasking
- Edge AI detection
- Hybrid underwater protocol
- MQTT remote telemetry

## Target Applications
- Naval hull inspection
- Subsea infrastructure monitoring
- Harbor security robotics
